    Shepherd

    Shepherd is an ArcCatalog-Style alternative to ArcGIS Server Services Directory.

    This is an Open Source project hosted on GitHub and developed by Amr Eldib.

    If you have any ideas, found bugs, suggestions or any feedback, please fill out this form, or submit an issue on GitHub.


    Built using jQuery, jsTree, Handlebars.js, ArcGIS API for Javascript, Highlightjs.org and bootstrap Flatly theme.
